### Usage

#### Bash/Zsh users

Source `.sh` file from `~/.bashrc` or `~/.zshrc`

#### Fish users

Run the snippet inside `.fish` file once.

#### Powershell users

Add the `.ps1` snippet to your `~/Documents/PowerShell/Microsoft.PowerShell_profile.ps1`
